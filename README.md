## 電子部品

- SMD: 贴片
- スルーホール: 直插
- コンデンサ: 电容
- セラミックコンデンサ: 瓷片电容
- レジスタ: 电阻
- ディスプレイ: 屏
- フラッシュ: 闪存
- タクタイルスイッチ: 轻触开关
- OLED ディスプレイ: OLED 显示屏
- MCU: 微控制器
- 32bit: 32 位
- ボックスピンヘッダ: 牛角插座
- ピンヘッダ: 排针
- ピンヘッダソケット:  排针插座
- I/O エキスパンダー: 扩展器芯片
- FCリボンケーブル: FC灰排线

- 新品: 全新原装
- 解体品: 拆机货
